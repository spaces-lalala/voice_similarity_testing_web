# Shadowing 語音相似度評分實驗

這是一個部署在 GitHub Pages 上的學術研究實驗網站，用於進行 shadowing 語音相似度評分實驗。

## 實驗說明

- **實驗類型**: Shadowing 語音相似度評分
- **機構**: NCCU人工智慧實務專題
- **區塊數量**: 10 個區塊
- **每區塊音檔對數**: 5 對
- **評分範圍**: 1-5 分 (1=完全不相似，5=完全相似)
- **休息時間**: 每區塊間 1 分鐘休息
- **預估時間**: 25-30 分鐘

✅ **專業的學術實驗界面**
- 清晰的實驗說明和評分標準

## 實驗流程

### 1. 歡迎與說明頁面
- 清晰的實驗說明和評分標準 (1-5分制)
- 參與者基本資訊填寫
- 知情同意書確認

### 2. 實驗進行頁面
- 音檔播放控制 (原始音檔 vs Shadowing音檔)
- 5分制相似度評分系統
- 實時進度顯示

### 3. 遊戲化休息頁面 🎮
- **載入動畫**: 模擬遊戲載入的進度條
- **實驗小知識**: 5個Shadowing技術相關知識
- **冷笑話時間**: 6個語言學研究室風格幽默
- **視覺效果**: 霓虹色彩、粒子動畫、發光效果
- **倒數計時**: 1分鐘休息時間，可跳過

### 4. 實驗完成頁面
- 完成統計顯示
- Google表單自動數據上傳
- 本地數據備份下載

詳細部署指南請參考 [`DEPLOYMENT.md`](DEPLOYMENT.md)

### 快速部署
1. **創建GitHub倉庫**並上傳所有文件
2. **啟用GitHub Pages** (Settings → Pages → Deploy from main branch)
3. **設定Google表單**收集實驗數據
4. **上傳音檔**到相應目錄 (50對音檔，共100個檔案)

## 注意事項

### ⚠️ 重要提醒
1. **音檔準備**: 確保所有音檔路徑正確且可訪問
2. **網路環境**: 實驗需要穩定網路來上傳數據  
3. **瀏覽器設定**: 建議在實驗前測試音頻播放
4. **數據備份**: 系統提供自動備份，建議定期下載
5. **隱私保護**: 遵守研究倫理和數據保護規範

### 🔧 故障排除
- **音檔無法播放**: 檢查檔案格式和路徑
- **評分無法提交**: 確認網路連線和表單設定
- **頁面顯示異常**: 清除瀏覽器快取重新載入
- **手機顯示問題**: 確認使用現代瀏覽器
