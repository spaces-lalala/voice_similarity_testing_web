# Shadowing 語音相似度評分實驗

這是一個部署在 GitHub Pages 上的學術研究實驗網站，用於進行 shadowing 語音相似度評分實驗。

## 實驗說明

- **實驗類型**: Shadowing 語音相似度評分
- **區塊數量**: 10 個區塊
- **每區塊音檔對數**: 5 對
- **評分範圍**: 1-7 分 (1=完全不相似，7=完全相似)
- **休息時間**: 每區塊間 1 分鐘休息
- **預估時間**: 25-30 分鐘

## 功能特色

✅ **專業的學術實驗界面**
- 清晰的實驗說明和評分標準
- 進度條顯示實驗進度
- 響應式設計，適配各種設備

✅ **完整的音頻控制**
- 個別播放原始音檔和 shadowing 音檔
- 順序播放兩段音檔功能
- 音頻重播控制

✅ **數據收集與管理**
- 整合 Google 表單收集數據
- 自動數據備份下載
- 完整的參與者資訊記錄

✅ **實驗流程控制**
- 自動區塊間休息計時
- 防止意外離開頁面
- 完整的實驗狀態管理

## 部署步驟

### 1. 創建 GitHub 倉庫
1. 在 GitHub 上創建新的倉庫
2. 將所有文件上傳到倉庫

### 2. 啟用 GitHub Pages
1. 進入倉庫設置頁面
2. 找到 "Pages" 設定
3. 選擇 "Deploy from a branch"
4. 選擇 "main" 分支和 "/ (root)" 目錄
5. 點擊 "Save"

### 3. 設定 Google 表單
1. 創建 Google 表單來收集實驗數據
2. 在 `js/experiment.js` 中更新 `googleFormUrl` 和 `formFields`
3. 確保表單欄位對應正確

### 4. 上傳音檔
1. 準備音檔文件（MP3 格式）
2. 按照區塊結構上傳到 `audio/` 目錄
3. 命名格式：`original_1.mp3`, `shadowing_1.mp3` 等

## 音檔準備指南

### 目錄結構
```
audio/
├── block1/
│   ├── original_1.mp3    # 區塊1-第1對-原始音檔
│   ├── shadowing_1.mp3   # 區塊1-第1對-shadowing音檔
│   ├── original_2.mp3
│   ├── shadowing_2.mp3
│   ├── ...
│   ├── original_5.mp3
│   └── shadowing_5.mp3
├── block2/
│   └── ... (同樣結構)
└── ...
├── block10/
    └── ... (同樣結構)
```

### 音檔要求
- **格式**: MP3 (推薦)
- **品質**: 適中品質即可 (128kbps - 320kbps)
- **長度**: 建議每段音檔不超過 30 秒
- **音量**: 統一標準化音量

## Google 表單設定

### 必要欄位
1. **參與者編號** (短答)
2. **年齡** (短答)
3. **性別** (選擇題)
4. **母語** (短答)
5. **區塊編號** (短答)
6. **音檔對編號** (短答)
7. **原始音檔路徑** (短答)
8. **Shadowing 音檔路徑** (短答)
9. **相似度評分** (短答)
10. **評分時間戳記** (短答)

### 取得表單提交 URL
1. 創建表單後，點擊右上角的「傳送」
2. 在彈出視窗中點擊「<>」(嵌入程式碼)
3. 複製 iframe 的 src 網址
4. 將 `/viewform` 替換為 `/formResponse`
5. 更新 `js/experiment.js` 中的 `googleFormUrl`

### 取得欄位名稱
1. 在表單編輯頁面，按 F12 開啟開發者工具
2. 切換到 Network 標籤
3. 填寫並提交表單
4. 在 Network 中找到 formResponse 請求
5. 查看 Form Data，複製 entry.xxxxxxxx 欄位名稱
6. 更新 `js/experiment.js` 中的 `formFields`

## 自訂設定

### 實驗參數調整
在 `js/experiment.js` 的 `EXPERIMENT_CONFIG` 中可以調整：
- `totalBlocks`: 總區塊數
- `pairsPerBlock`: 每區塊音檔對數
- `breakDuration`: 休息時間（秒）

### 樣式自訂
在 `css/styles.css` 中可以自訂：
- 顏色主題
- 字體大小
- 布局樣式
- 響應式斷點
